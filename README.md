Felsefe hakkında bir web sitesi içinde felsefenin ilkeleri türleri ve belli başlı adı fazlaca duyulan, çoğunluğun bildiği filozoflardan oluşan ve tıklandığında çeşitlibilgiler web sitelerine gönderen linklerden oluşuyor. İlk filozof olarak adlandırılan Thales hakkında bilgi verildi nasıl filozof olarak adlandırıldığı sürecine değinildi. Aynı zamanda felsefe ve filozofun etimolijisine dair bilgiler bulunur. İki filozofun eserlerinr tablo şeklinde değinildi. Felsefe odaklı filozoflara değinilen html işaretleme dili kullanılarak yapılan felsefe+filozoflardan oluşan bir web sitesidir
